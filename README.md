# Weather Forecasting Application

## Description

The Weather Forecasting Application is a web-based tool built with Flask that allows users to get real-time weather updates and forecasts for any location. Users can search for a city and receive current weather conditions, hourly forecasts, and daily predictions.

## Features

- **Real-time Weather Updates**: Get current weather conditions for any city.
- **Detailed Forecasts**: Access hourly and daily forecasts.
- **User-Friendly Interface**: Simple and intuitive design for easy navigation.
- **Responsive Design**: Works seamlessly on both desktop and mobile devices.

## Technologies Used

- **Backend**: Flask
- **Frontend**: HTML, CSS, JavaScript
- **APIs**: [OpenWeatherMap API](https://openweathermap.org/api) for weather data
- **Database**: SQLite (or any other database of your choice)

## Installation

### Prerequisites

- Python 3.x
- Pip

### Steps to Install

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/weather-forecasting-app.git
   cd weather-forecasting-app
   ```

2. Create a virtual environment (optional but recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Set up your API key:
   - Sign up at [OpenWeatherMap](https://openweathermap.org/) and get your API key.
   - Add the API key to your environment variables or a configuration file.

5. Run the application:
   ```bash
   python app.py
   ```

6. Open your web browser and go to `http://127.0.0.1:5000/` to view the application.

## Usage

1. Enter the name of the city in the search bar.
2. Click the "Get Weather" button.
3. View the current weather, hourly forecasts, and daily predictions.
